# Cognizant Deep Skilling Program – .NET Full Stack Engineer Track

Welcome to my learning repository for the **Cognizant Deep Skilling Program (Digital Nurture 5.0)**. This repository serves as a centralized collection of notes, hands-on exercises, assignments, and project work completed throughout the program.

The training focuses on developing industry-ready skills in **.NET Full Stack Development**, covering backend technologies, frontend frameworks, cloud-native development, DevOps practices, and Generative AI fundamentals.

---

## Program Overview

The Cognizant Deep Skilling Program is designed to provide practical exposure to modern software development technologies and industry best practices. Through structured learning modules and hands-on exercises, this program strengthens problem-solving abilities and full-stack development expertise.
